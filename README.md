# 📷 VINTAGE-X

A retro-styled, browser-based photo editor with filters, effects, and adjustments, no installs, no uploads to a server, everything runs locally in your browser.

## Features

- **Filters** — Nashville, Sepia, Chief Keef, Nuke, Phreshboy, 2014, $$$, Pandora, Bleach Bypass, Stressor, B&W, Desat, and more
- **Effects** — Lo-Fi, VHS Glitch, Polaroid, Film Grain, Vintage, Fade
- **Adjust** — Brightness, Contrast, Saturation, Warmth, Tint
- **Blur / Pixelate / Vignette** — Gaussian blur, pixel art effect, vignette overlay
- **Collage** — Arrange multiple images in 2x2, 3x1, 1x3, 2x1, 1x2, or 3x3 layouts
- **Text Overlay** — Add custom text with size, color, position, shadow, outline, and caps options
- **Drag & Drop** — Drop images directly onto the canvas
- **Download** — Export your edited image as a PNG
- **Random** — Apply a random filter combo with one click
- **Reset** — Restore all settings to default

## How to Use

1. Open the index in any modern browser
2. Click **SELECT IMAGE** or drag and drop a photo onto the canvas
3. Pick a filter, adjust sliders, add text
4. Hit **Download** to save your edit

## Tech Stack

- Pure HTML, CSS, and JavaScript, zero dependencies
- Canvas API for image processing
- Retro Windows 95-inspired UI
